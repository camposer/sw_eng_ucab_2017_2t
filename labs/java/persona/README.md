# Ejercicio Persona

Este ejercicio presenta un CRUD sobre una entidad llamada persona

# Configuración del JRE

Windows -> Preferences -> Installed JREs -> 
Seleccionar carpeta JDK (C:\Archivos de programa\Java\jdk) ->
Finish -> Seleccionar

# Creación de la BD

Ejecutar el script adjunto `persona.sql`

# Configuración del conector de MySQL

Clic derecho sobre el proyecto ->
Properties -> Java Build Path ->
Libraries -> Add JAR ->
Seleccionar persona/lib/mysql-connector.jar
